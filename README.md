### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:erickhansx/blog-app.git
```

### Install

Install this project with:

```sh
  cd my-project
  gem install
  bundle install
```

### Usage

To run the project, execute the following command:

